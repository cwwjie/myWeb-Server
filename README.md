## <div id="00">操作说明</div>

* 项目初始化项目  
```
npm install
```

* 测试环境
	* 启动项目  
	```
	supervisor server.js   
	```
<br><br>
* 生产环境
	* 不需要打包 *直接进行上传就可*  
	```
	forever server.js   
	```
<br><br>
